# Simple-Calculator-
A simple calculator for two numbers made with python 

# How To Run The Calculator:

1. On your computer navigate to the folder containing the file and run
```bash
python Simple-Calculator.py
```
                        (or)
2. Copy the code in GitHub Codespaces or in any free external website (like, for eg: kaggle kernels)
  
# How To Use The Calculator:

1. After running the code the computer will give you a text-terminal in which you have to enter you first number
2. After entering the first number you will have to enter the second number in the second text-terminal.
3. The computer will automatically give you the sum, difference, product and quotient of the two numbers

# Features:

1. Easy to use with clean text interface 
2. Gives four answers at once
3. Best for simple calculations
4. The code is written in python
